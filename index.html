How to update the catalog permanently:

  1. Make all your changes in the browser (add, edit, delete items).
  2. Click the "Download Database" button to save the updated products.json file.
  3. In your project folder, replace the old products.json with the new one you just downloaded.
  4. Commit and push the changes to your GitHub repository to make them live.
